# SoulmaskDB

Soulmask data viewer

## 한국어

Soulmask 게임 데이터 뷰어입니다.  
GitHub Pages에서 실행할 수 있도록 만든 정적 웹페이지이며, `Game.po`, `Game_en.po`, `tianfuzongbiaoxin2.json`, 그리고 아이콘 리소스 폴더를 이용해 캐릭터 특성 정보를 보기 쉽게 표시합니다.

### 주요 기능

- `Game.po` / `Game_en.po` 자동 로딩
- `DT_GiftZongBiao` 전용 보기
- `캐릭터특성 보기` 버튼 지원
- JSON 기준 상세 분류
- 출처 한글 표시
- 아이콘 자동 표시

### 폴더 구조

```text
/
├─ index.html
├─ .nojekyll
├─ data/
│  ├─ Game.po
│  ├─ Game_en.po
│  └─ tianfuzongbiaoxin2.json
└─ assets/
   └─ Game/
      └─ UI/
         └─ resource/
```
